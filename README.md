# 📝 Grievance System

A web-based grievance management system that allows users to submit
complaints and enables administrators to track, manage, and resolve them
efficiently.

------------------------------------------------------------------------

## 🚀 Features

-   👤 User Registration & Login\
-   📝 Submit Grievances\
-   📊 Track Complaint Status\
-   🛠️ Admin Dashboard\
-   ✅ Resolve / Reject Complaints\
-   📂 Organized Complaint Records

------------------------------------------------------------------------

## 🛠️ Tech Stack

-   Frontend: HTML, CSS, JavaScript\
-   Backend: Node.js\
-   Database: MongoDB / MySQL / PostgreSQL\
-   Version Control: Git & GitHub

------------------------------------------------------------------------

## 📁 Project Structure

grievance-system/ │── frontend/ │── backend/ │── routes/ │──
controllers/ │── models/ │── database/ │── README.md

------------------------------------------------------------------------

## ⚙️ Installation & Setup

git clone https://github.com/yash020/grievance-system.git\
cd grievance-system\
npm install

------------------------------------------------------------------------

## ▶️ How to Run the Project (For Interviewer)

1.  Clone repo\

2.  Install dependencies\

3.  Create .env file with: PORT=5000\
    DB_URI=your_database_url\
    SECRET_KEY=your_secret_key

4.  Run: npm start

5.  Open: http://localhost:5000
